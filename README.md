# .github

This repository powers the [Nuvio Software](https://github.com/Nuvio-Software) GitHub organisation profile.

- `profile/README.md` — rendered on https://github.com/Nuvio-Software as the public org landing page.

Add shared community health files here as needed (e.g. `CODE_OF_CONDUCT.md`, `SECURITY.md`, issue/PR templates) — GitHub will use them as defaults across repos in the org that don't define their own.
